# GMS Homecoming 2026

Early GitHub Pages landing page for a proposed Geylang Methodist School Homecoming 2026 site.

## What is included
- Static landing page (`index.html`, `styles.css`, `script.js`)
- GitHub Pages workflow deployment
- Research-grounded tone and copy framework
- Honest placeholders for details not yet confirmed

## What is already grounded
Publicly visible sources used for the first draft:
1. https://www.gmsalumni.net
2. https://www.gmsalumni.net/event-details-registration/gms-alumni-101st-founders-day-lunch
3. https://geylangmethodistpri.moe.edu.sg/about-us/our-partners/alumni

Signals taken from those sources:
- School/alumni motto: "To Grow in Friendship, Body and Mind"
- Alumni themes: friendship, belonging, service, outreach, gratefulness, community engagement
- Existing alumni events use reunion + meal + founder's-day framing

## What still needs organiser confirmation
- Official event name formatting
- Exact event date and time
- Venue
- Registration / payment link
- Sponsor or committee contact details
- Approved photos / logos / crest usage

## Local preview
Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

## Deployment
This repo is configured for GitHub Pages via GitHub Actions.
